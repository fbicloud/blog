windows系统的换行方式CRLF: "\r\n"

Linux系统的换行方式LF: "\n"

在你使用git的时候，git会自动将代码当中与你当前系统不同的换行方式转化成你当前系统的换行方式，从而造成冲突。

解决方法

1

git config --global core.autocrlf false
